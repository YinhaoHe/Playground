## Boilerplate

html5

```html
<!DOCTYPE html>
<html>
  <head>
    <meta charset="utf-8">
    <title></title>
  </head>
  <body>
    
  </body>
</html>
```

- title: website tab's name
- charset: utf-8 encoding
- meta: `<meta name="description" content="How to search this website">` - used for search engine
